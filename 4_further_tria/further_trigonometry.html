Further Trigonometry — 핵심개념 (A-Level P3 & P4)

Angle Sum “SC+CS / CC−SS” sin은 교차쌍·부호 그대로 / cos은 동일쌍·부호 뒤집힘 / tan은 분모 부호 반대
cos2x 세 형태 없애고 싶은 항으로 선택 \(\cos^2x\) 없애기 → \(1-2\sin^2x\) / \(\sin^2x\) 없애기 → \(2\cos^2x-1\)
sec/csc/cot 미분 “co- 붙으면 음수” + 쌍 tan↔︎sec, cot↔︎csc / sec·csc는 자기 자신 생존
Sum-to-Product 부호 따라 앞 함수 결정 \(+\)이면 앞 함수 유지 / \(-\)이면 sin↔︎cos 교체 / cos−cos만 전체 음수
Compound 직각삼각형 시각화 \(a, b\) = 두 변, \(R=\sqrt{a^2+b^2}\) = 빗변, \(\tan\alpha = b/a\)
해 여러 개 θ=nx로 치환 후 범위 확장 \(x\in[0,2\pi]\)이면 \(\theta=nx\in[0,2n\pi]\) → 주기마다 +2π 반복

1. Basic Identity (when angles are the same)

# Formula Note
1 \(\cos^2 x + \sin^2 x = 1\) Pythagorean identity
2 \(1 + \tan^2 x = \sec^2 x\) Dividing by \(\cos^2 x\)
3 \(\cot^2 x + 1 = \csc^2 x\) Dividing by \(\sin^2 x\)

Tip — “3-from-1” chain: 공식 ①만 외우면 나머지는 유도. ① \(\div \cos^2 x\) → ② / ① \(\div \sin^2 x\) → ③ 외울 것은 사실상 \(\sin^2 x + \cos^2 x = 1\) 하나뿐.

2. Angle Sum and Difference Identity

# Formula Note
1 \(\sin(x+y) = \sin x \cos y + \cos x \sin y\)
2 \(\sin(x-y) = \sin x \cos y - \cos x \sin y\) substitute \(y \to -y\) in (1)
3 \(\cos(x+y) = \cos x \cos y - \sin x \sin y\)
4 \(\cos(x-y) = \cos x \cos y + \sin x \sin y\) substitute \(y \to -y\) in (3)
5 \(\tan(x+y) = \dfrac{\tan x + \tan y}{1 - \tan x \tan y}\) divide (1)÷(3) by \(\cos x \cos y\)
6 \(\tan(x-y) = \dfrac{\tan x - \tan y}{1 + \tan x \tan y}\) substitute \(y \to -y\) in (5)

Tip — “SC+CS / CC−SS” 니모닉:

공식 암기 코드 부호
\(\sin(A+B)\) \(\sin A\cos B + \cos A\sin B\) SC + CS ± 그대로
\(\cos(A+B)\) \(\cos A\cos B - \sin A\sin B\) CC − SS ± 반대
\(\tan(A+B)\) \(\dfrac{\tan A + \tan B}{1 - \tan A\tan B}\) 분자=합, 분모=1∓곱 분모 부호 반대
  • sin: 서로 다른 쌍 (S·C + C·S), 부호는 ± 그대로
  • cos: 같은 쌍 (C·C − S·S), 부호는 뒤집힘 → cos(+)는 빼고, cos(−)는 더함
  • tan: \(A-B\) 버전은 분자는 빼고, 분모는 1+곱 (분모 부호만 반대)

Why does “substitute \(y \to -y\)” work? \(\sin\) and \(\tan\) are odd functions; \(\cos\) is an even function: \[\sin(-y) = -\sin y, \qquad \cos(-y) = \cos y, \qquad \tan(-y) = -\tan y\] So replacing \(y\) with \(-y\) in formula (1) flips the sign of every \(\sin y\) term but leaves \(\cos y\) unchanged — giving formula (2) directly. The same logic applies to (3)→(4) and (5)→(6).

(A) Double Angle Formulae

“let \(y = x\)”의 의미: Section 2의 Angle Sum 공식에서 \(y\) 자리에 \(x\)를 그대로 대입하면 \(\sin(x+x)\), \(\cos(x+x)\), \(\tan(x+x)\) 가 되고, 이것이 곧 Double Angle 공식이 됩니다. 즉 새로운 공식이 아니라 Angle Sum의 특수 케이스입니다.

유도 과정:

\[\sin(x+y)\big|_{y=x} \;=\; \sin x\cos x + \cos x\sin x \;\Rightarrow\; \boxed{\sin 2x = 2\sin x \cos x}\]

\[\cos(x+y)\big|_{y=x} \;=\; \cos x\cos x - \sin x\sin x \;\Rightarrow\; \cos 2x = \cos^2 x - \sin^2 x\]

cos2x 세 가지 형태 (Basic Identity \(\cos^2x = 1-\sin^2x\) 또는 \(\sin^2x = 1-\cos^2x\) 대입):

\[\boxed{\cos 2x = \cos^2 x - \sin^2 x = 1 - 2\sin^2 x = 2\cos^2 x - 1}\]

Tip — cos2x 세 형태 선택 가이드:

상황 사용할 형태
\(\sin^2 x\)와 \(\cos^2 x\) 둘 다 있음 \(\cos^2 x - \sin^2 x\)
\(\cos^2 x\)를 없애고 싶음 (sin만 남기기) \(1 - 2\sin^2 x\)
\(\sin^2 x\)를 없애고 싶음 (cos만 남기기) \(2\cos^2 x - 1\)

역방향: \(\sin^2 x = \dfrac{1-\cos 2x}{2}\), \(\cos^2 x = \dfrac{1+\cos 2x}{2}\) → 적분할 때 필수 (Half Angle)

\[\tan(x+y)\big|_{y=x} \;=\; \frac{\tan x+\tan x}{1-\tan x\tan x} \;\Rightarrow\; \boxed{\tan 2x = \frac{2\tan x}{1 - \tan^2 x}}\]

왜 유용한가? 방정식에 \(\sin x\)와 \(\sin 2x\) (또는 \(\cos 2x\))가 동시에 섞여 있을 때, Double Angle로 \(2x\) 항을 \(x\)로 바꿔 각도를 통일한 뒤 인수분해(factorise)할 수 있습니다. 예: \(\sin 2x = \sin x\) → \(2\sin x\cos x - \sin x = 0\) → \(\sin x(2\cos x - 1) = 0\)

(B) Half Angle Formulae (Power Reduction Formulae)

Derivation: Rearrange the two forms of \(\cos 2x\) directly:

\[\cos 2x = 1 - 2\sin^2 x \;\Rightarrow\; \sin^2 x = \frac{1 - \cos 2x}{2}\]

\[\cos 2x = 2\cos^2 x - 1 \;\Rightarrow\; \cos^2 x = \frac{1 + \cos 2x}{2}\]

When to use: Any time you see a squared trig term — it blocks direct integration or factorisation. Replace it with a \(\cos 2x\) expression to reduce the power.

  • Integration: \(\displaystyle\int \sin^2 x \, dx = \int \frac{1-\cos 2x}{2} \, dx = \frac{x}{2} - \frac{\sin 2x}{4} + C\)
  • Equation: \(\cos 2x + \sin^2 x = 0 \;\to\; (1-2\sin^2 x) + \sin^2 x = 0 \;\to\; \sin^2 x = 1\)

(C) Triple Angle Formulae — let \(y = 2x\)

\[\sin 3x = 3\sin x - 4\sin^3 x\]

\[\cos 3x = 4\cos^3 x - 3\cos x\]

Derivation: Expand \(\sin(2x+x)\) using Angle Sum, then substitute Double Angle results:

\[\sin 3x = \sin 2x\cos x + \cos 2x\sin x = 2\sin x\cos^2 x + (1-2\sin^2 x)\sin x = 3\sin x - 4\sin^3 x\]

Likewise \(\cos 3x = \cos 2x\cos x - \sin 2x\sin x = (2\cos^2 x-1)\cos x - 2\sin^2 x\cos x = 4\cos^3 x - 3\cos x\).

When to use: - Forward (\(3x \to x\)): equation mixes \(\sin 3x\) and \(\sin x\) — replace \(\sin 3x\) to unify the angle.
e.g. \(\sin 3x = \sin x \;\to\; 3\sin x - 4\sin^3 x = \sin x \;\to\; \sin x(1-2\sin^2 x)=0\) - Reverse (\(x \to 3x\)): expression contains \(\sin^3 x\) or \(\cos^3 x\) — write it as a single trig term for integration or simplification.
e.g. \(\cos^3 x = \tfrac{1}{4}(\cos 3x + 3\cos x)\)

(D) Compound Formulae

“Reverse of Angle Sum”의 의미: Angle Sum 공식을 정방향(→) 으로 쓰면 합친 형태를 전개(expand)하는 것이고, Compound는 이것을 역방향(←) 으로 — 즉 분리된 \(a\sin x + b\cos x\)를 하나의 항으로 합치는 것입니다.

유도 과정 (\(a\sin x + b\cos x\) 의 경우):

Angle Sum 공식: \(\sin(x+\alpha) = \sin x\cos\alpha + \cos x\sin\alpha\)

양변에 \(R\)을 곱하면:

\[R\sin(x+\alpha) = \underbrace{R\cos\alpha}_{=\,a}\sin x + \underbrace{R\sin\alpha}_{=\,b}\cos x\]

따라서 \(a = R\cos\alpha\), \(b = R\sin\alpha\) 로 놓으면:

\[\boxed{a\sin x + b\cos x = R\sin(x+\alpha)}\]

\[R = \sqrt{a^2+b^2}, \qquad \tan\alpha = \frac{b}{a} \quad (\alpha \in [0°,\,90°])\]

Tip — 직각삼각형으로 시각화:

\(a\sin x + b\cos x\)에서 \(a\)와 \(b\)를 직각삼각형의 두 변으로 놓으면:

         R = √(a²+b²)  ← 빗변
        /|
       / |  b  (cos 계수)
      /α |
     ─────
      a  (sin 계수)
  • \(R\) = 빗변 = 진폭 (amplitude)
  • \(\tan\alpha = \dfrac{b}{a}\) → \(\alpha\) = 위상(phase shift)
  • \(a, b\) 부호 그대로 삼각형에 넣고 → 사분면으로 \(\alpha\) 범위 결정

같은 방식으로 (\(a\cos x + b\sin x\) 의 경우):

Angle Difference 공식: \(\cos(x-\alpha) = \cos x\cos\alpha + \sin x\sin\alpha\)

\[R\cos(x-\alpha) = \underbrace{R\cos\alpha}_{=\,a}\cos x + \underbrace{R\sin\alpha}_{=\,b}\sin x\]

\[\boxed{a\cos x + b\sin x = R\cos(x-\alpha)}\]

왜 유용한가? \(a\sin x + b\cos x = k\) 처럼 sin과 cos이 섞인 방정식은 직접 풀기 어렵지만, \(R\sin(x+\alpha) = k\) 형태로 바꾸면 \(\sin\theta = k/R\) 로 바로 풀 수 있습니다.

(E) Sum-to-Product Identity

유도 방법: Angle Sum/Difference 공식에서 \(x+y = P\), \(x-y = Q\) 로 치환하면 \(x = \frac{P+Q}{2}\), \(y = \frac{P-Q}{2}\) 가 됩니다. 이것을 대입해 정리하면 아래 공식들이 나옵니다.

\[\sin P + \sin Q = 2\sin\!\left(\frac{P+Q}{2}\right)\cos\!\left(\frac{P-Q}{2}\right)\]

\[\sin P - \sin Q = 2\cos\!\left(\frac{P+Q}{2}\right)\sin\!\left(\frac{P-Q}{2}\right)\]

\[\cos P + \cos Q = 2\cos\!\left(\frac{P+Q}{2}\right)\cos\!\left(\frac{P-Q}{2}\right)\]

\[\cos P - \cos Q = -2\sin\!\left(\frac{P+Q}{2}\right)\sin\!\left(\frac{P-Q}{2}\right)\]

Tip — Sum-to-Product 패턴:

형태 결과 패턴
\(\sin P + \sin Q\) \(2\sin\!\left(\frac{P+Q}{2}\right)\cos\!\left(\frac{P-Q}{2}\right)\) sin+sin → sin·cos
\(\sin P - \sin Q\) \(2\cos\!\left(\frac{P+Q}{2}\right)\sin\!\left(\frac{P-Q}{2}\right)\) sin−sin → cos·sin (앞뒤 바뀜)
\(\cos P + \cos Q\) \(2\cos\!\left(\frac{P+Q}{2}\right)\cos\!\left(\frac{P-Q}{2}\right)\) cos+cos → cos·cos
\(\cos P - \cos Q\) \(-2\sin\!\left(\frac{P+Q}{2}\right)\sin\!\left(\frac{P-Q}{2}\right)\) cos−cos → −sin·sin ← 유일하게 음수!
  • 앞 인수 → \(\dfrac{P+Q}{2}\) (평균), 뒤 인수 → \(\dfrac{P-Q}{2}\) (반차)
  • 같은 부호(+): 앞 함수 유지 / 다른 부호(−): 앞 함수가 바뀜 (sin↔︎cos)
  • cos−cos만 전체에 음수 붙음 — 특이 케이스로 별도 기억

왜 유용한가? \(\sin P + \sin Q = 0\) 처럼 서로 다른 두 각도의 합/차가 있을 때, 이를 곱(product) 형태로 바꿔 인수분해할 수 있습니다. 곱 = 0 이면 각 인수가 0인 경우로 나눠서 풀 수 있기 때문입니다. 예: \(\sin 3x + \sin x = 0\) → \(2\sin 2x \cos x = 0\) → \(\sin 2x = 0\) 또는 \(\cos x = 0\)

3. Solving Trig Equations — Finding All Roots in [0, 2π]

언제 쓰나? 삼각함수 값이 \(\frac{4}{3}\), \(-2\) 처럼 “표에 없는 값”일 때, 정확한 각도를 숫자로 쓸 수 없으므로 역삼각함수 표현 그대로 답을 쓴다. sec / csc / cot 가 나오면 먼저 역수(reciprocal)로 변환한 뒤 아래 절차를 따른다.

Step 0 — sec / csc / cot 가 나오면 먼저 변환

\[\sec x = k \;\Rightarrow\; \cos x = \frac{1}{k}\] \[\csc x = k \;\Rightarrow\; \sin x = \frac{1}{k}\] \[\cot x = k \;\Rightarrow\; \tan x = \frac{1}{k}\]

변환한 뒤 Step 1부터 동일하게 진행한다.

주의: \(|k| < 1\) 이면 \(|\frac{1}{k}| > 1\) 이 되어 sec/csc 는 \([-1, 1]\) 범위를 벗어날 수 있다. sin/cos 의 범위는 \([-1, 1]\) 이므로 \(|\frac{1}{k}| > 1\) 이면 해가 존재하지 않는다.

Step 1 — 기준각(Reference Angle) 구하기

핵심 개념: 기준각이란? 기준각 \(\alpha\)는 항상 양수, \(0 \leq \alpha \leq \frac{\pi}{2}\) 사이의 각도다. 단위원에서 각도 \(x\)와 가장 가까운 x축 사이의 각도 거리를 말한다.

왜 부호를 제거하는가?

\(\tan x = -2\) 일 때 계산기에 \(\arctan(-2)\)를 입력하면 \(-1.107\) rad 이 나온다. 이것은 \(\left[-\frac{\pi}{2},\, \frac{\pi}{2}\right]\) 범위의 값이라 \([0, 2\pi]\) 해와 직접 연결이 안 된다.

대신: \[\text{"tan의 크기가 2인 기준각"} \;\Rightarrow\; \alpha = \arctan(2) = 1.107 \text{ rad}\]

이 \(\alpha\)는 항상 1사분면에 존재한다. 이후 부호는 사분면 선택이 담당한다.

한 줄 요약: 기준각 \(\alpha\) = 크기(magnitude) 담당 / 부호 = 사분면 담당

각 함수별 기준각:

함수 방정식 기준각 \(\alpha\)
sin \(\sin x = k\) \(\arcsin\|k\|\)
cos \(\cos x = k\) \(\arccos\|k\|\)
tan \(\tan x = k\) \(\arctan\|k\|\)
csc \(\sin x = \frac{1}{k}\) \(\arcsin\left\|\frac{1}{k}\right\|\)
sec \(\cos x = \frac{1}{k}\) \(\arccos\left\|\frac{1}{k}\right\|\)
cot \(\tan x = \frac{1}{k}\) \(\arctan\left\|\frac{1}{k}\right\|\)

Step 2 — 부호에 따른 사분면 파악 (ASTC)

          sin +
     2사분면 | 1사분면
  sin+,cos- | sin+,cos+   → tan-  |  tan+
  ──────────┼──────────
  sin-,cos- | sin-,cos+   → tan+  |  tan-
     3사분면 | 4사분면
          sin -

외우는 법 — “All Students Take Calculus” (1→2→3→4사분면 순): - All (1사분면): sin, cos, tan 모두 양수 → csc, sec, cot 도 모두 양수 - Students (2사분면): sin만 양수 → csc도 양수 - Take (3사분면): tan만 양수 → cot도 양수 - Calculus (4사분면): cos만 양수 → sec도 양수

sec, csc, cot 는 각각 cos, sin, tan 의 역수이므로 부호가 항상 같다.

Step 3 — [0, 2π] 범위의 해 조립

기준각 \(\alpha \in \left[0, \frac{\pi}{2}\right]\) 로 놓으면:

sin x = k (또는 csc x = k → sin x = 1/k)

k의 부호 해당 사분면 해
\(k > 0\) 1사분면, 2사분면 \(x = \alpha, \quad x = \pi - \alpha\)
\(k < 0\) 3사분면, 4사분면 \(x = \pi + \alpha, \quad x = 2\pi - \alpha\)

cos x = k (또는 sec x = k → cos x = 1/k)

k의 부호 해당 사분면 해
\(k > 0\) 1사분면, 4사분면 \(x = \alpha, \quad x = 2\pi - \alpha\)
\(k < 0\) 2사분면, 3사분면 \(x = \pi - \alpha, \quad x = \pi + \alpha\)

tan x = k (또는 cot x = k → tan x = 1/k)

k의 부호 해당 사분면 해
\(k > 0\) 1사분면, 3사분면 \(x = \alpha, \quad x = \pi + \alpha\)
\(k < 0\) 2사분면, 4사분면 \(x = \pi - \alpha, \quad x = 2\pi - \alpha\)

왜 항상 해가 2개인가? 단위원에서 같은 \(y\)값(sin), \(x\)값(cos), 기울기(tan)를 갖는 점은 대칭적으로 정확히 2개 존재하기 때문이다. (tan은 주기가 π이므로 \([0,2\pi]\)에서 정확히 2개.)

한눈에 보는 해 공식 지도

              π/2
               │
  (2사분면)    │    (1사분면)
  π − α  ━━━━━┿━━━━━  α         sin+, cos-, tan-  |  sin+, cos+, tan+
               │
  ─────────────┼─────────────
               │
  (3사분면)    │    (4사분면)
  π + α  ━━━━━┿━━━━━  2π − α    sin-, cos-, tan+  |  sin-, cos+, tan-
               │
              3π/2

  패턴 요약:
  • 1사분면 →  α
  • 2사분면 →  π − α
  • 3사분면 →  π + α
  • 4사분면 →  2π − α

완전한 예시

예시 1: \(\tan x = \frac{4}{3}\), \(\tan x = -2\), 범위 \([0, 2\pi]\)

① \(\tan x = \frac{4}{3}\) (양수) - 기준각: \(\alpha = \arctan\!\left(\frac{4}{3}\right) \approx 0.927\) rad - tan > 0 → 1사분면, 3사분면 - 해: \(x = \arctan\dfrac{4}{3}, \quad x = \pi + \arctan\dfrac{4}{3}\)

② \(\tan x = -2\) (음수) - 기준각: \(\alpha = \arctan(2) \approx 1.107\) rad - tan < 0 → 2사분면, 4사분면 - 해: \(x = \pi - \arctan 2, \quad x = 2\pi - \arctan 2\)

\[\boxed{x = \arctan\tfrac{4}{3},\quad \pi - \arctan 2,\quad \pi + \arctan\tfrac{4}{3},\quad 2\pi - \arctan 2}\]

예시 2: \(\sec x = -\frac{5}{3}\), 범위 \([0, 2\pi]\)

  • Step 0: \(\cos x = -\dfrac{3}{5}\) (역수로 변환)
  • 기준각: \(\alpha = \arccos\!\left(\dfrac{3}{5}\right)\)
  • cos < 0 → 2사분면, 3사분면
  • 해: \(x = \pi - \arccos\dfrac{3}{5}, \quad x = \pi + \arccos\dfrac{3}{5}\)

예시 3: \(\cot x = \frac{1}{2}\), 범위 \([0, 2\pi]\)

  • Step 0: \(\tan x = 2\) (역수로 변환)
  • 기준각: \(\alpha = \arctan(2)\)
  • tan > 0 → 1사분면, 3사분면
  • 해: \(x = \arctan 2, \quad x = \pi + \arctan 2\)

4. 해가 2개보다 많을 때 — Multiple Solutions

언제 발생하는가? 방정식에 \(\sin(2x)\), \(\cos(3x)\), \(\tan(4x)\) 처럼 배수 각도(multiple angle) 가 나오면, \([0, 2\pi]\) 범위에서 해가 2개보다 훨씬 많이 나온다.

핵심 아이디어 — θ = nx 치환 후 범위 확장

\[x \in [0,\, 2\pi] \;\Rightarrow\; \theta = nx \in [0,\, 2n\pi]\]

  • \(x\) 범위를 그대로 쓰지 말고, 치환한 θ의 새 범위로 확장한다.
  • 새 범위 \([0, 2n\pi]\) 안에서 기준각에 \(2\pi\)를 반복해서 더하며 모든 해를 나열한다.
  • 마지막에 \(n\)으로 나눠서 \(x\)로 환원한다.

Step-by-step 절차

예) \(\sin(2x) = \dfrac{1}{2}\), \(\quad x \in [0, 2\pi]\)

① θ 치환 및 범위 설정

\[\theta = 2x \;\Rightarrow\; \theta \in [0,\, 4\pi]\]

② 기준각 구하기

\[\alpha = \arcsin\!\left(\frac{1}{2}\right) = \frac{\pi}{6}\]

③ 한 주기 \([0, 2\pi]\) 안의 해 (sin > 0 → 1·2사분면)

\[\theta = \frac{\pi}{6}, \quad \frac{5\pi}{6}\]

④ 다음 주기 \([2\pi, 4\pi]\) — 각각에 \(2\pi\) 더하기

\[\theta = \frac{\pi}{6} + 2\pi = \frac{13\pi}{6}, \quad \frac{5\pi}{6} + 2\pi = \frac{17\pi}{6}\]

⑤ \(x = \theta \div 2\)

\[x = \frac{\pi}{12},\quad \frac{5\pi}{12},\quad \frac{13\pi}{12},\quad \frac{17\pi}{12}\]

총 해의 수 예측: \(\sin(nx) = k\)는 \([0, 2\pi]\)에서 보통 \(2n\)개 (단, 경계값이나 특수값은 더 적을 수 있음)

해의 수 예측표

방정식 치환 범위 예상 해의 수
\(\sin x = k\) — \([0, 2\pi]\) 2개
\(\sin 2x = k\) \(\theta=2x\) \([0, 4\pi]\) 4개
\(\sin 3x = k\) \(\theta=3x\) \([0, 6\pi]\) 6개
\(\sin 4x = k\) \(\theta=4x\) \([0, 8\pi]\) 8개
\(\sin nx = k\) \(\theta=nx\) \([0, 2n\pi]\) \(2n\)개

주의: \(\sin(nx) = 0\) 또는 \(\sin(nx) = \pm1\) 처럼 경계값이면 겹치는 해가 생겨 실제 해 수가 다를 수 있다. (Q10의 sin 4x = 0 → 9개가 나오는 것처럼)

Compound 공식 후 범위 이동 케이스

예) \(\sin\!\left(x - \dfrac{\pi}{3}\right) = \dfrac{1}{2}\), \(\quad x \in [0, 2\pi]\)

치환: \(u = x - \dfrac{\pi}{3}\)이면, \(x \in [0, 2\pi]\)에서

\[u \in \left[-\frac{\pi}{3},\; \frac{5\pi}{3}\right]\]

이 이동된 범위 안에서 모든 해를 찾아야 한다. 기준각 \(\alpha = \pi/6\)이므로:

후보 범위 안?
\(u = \pi/6\) ✓
\(u = \pi - \pi/6 = 5\pi/6\) ✓
\(u = -\pi + \pi/6 = -5\pi/6\) ✗ (범위 밖)

→ \(x = u + \pi/3\)으로 환원: \(x = \pi/2,\; 7\pi/6\)

핵심: Compound 치환 후에는 반드시 범위를 새로 계산하고, 그 범위에서 모든 후보를 검토해야 한다.

KEY — Strategy for Solving Trig Equations

TipKEY — Therefore, when solving trigonometric equations:
What you see Action Formula
The same angle throughout Use directly — replace one squared term with another Basic Identities
Mixed angles \(X\) and \(2X\) Make them the same angle (A) Double Angle or (B) Power Reduction
Mixed angles \(P\) and \(Q\) (unrelated) Convert sum/difference to a product, then factorise (E) Sum-to-Product
\(a\sin x + b\cos x = 0\) Rearrange: divide through by \(\cos x\) \(\tan x = -\dfrac{a}{b}\)
\(a\sin x + b\cos x \neq 0\) Compound into a single term \(R\sin(x \pm \alpha)\) (D) Compound
