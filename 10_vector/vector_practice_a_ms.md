## Question 1 &nbsp;&nbsp; (9 marks)

**Line 1:** $\mathbf{r} = (0,7,-4) + s(4,-3,1)$ &ensp; **Line 2:** $\mathbf{r} = (-7,1,8) + t(-3,0,2)$

### Part (a) — 5 marks

Set components equal:

| Component | Equation |
|-----------|----------|
| **i** | $4s = -7 - 3t$ |
| **j** | $7 - 3s = 1$ |
| **k** | $-4 + s = 8 + 2t$ |

From **j**: $3s = 6 \Rightarrow s = 2$ &emsp; **M1 A1**

Substitute into **k**: $-4 + 2 = 8 + 2t \Rightarrow t = -5$ &emsp; **A1**

Verify **i**: $4(2) = 8$, $\;-7-3(-5) = -7+15 = 8$ ✓ &emsp; **M1** (must state check)

Position vector: $(0+8,\;7-6,\;-4+2) = \mathbf{8i + j - 2k}$ &emsp; **A1**

### Part (b) — 4 marks

Direction vectors: $\mathbf{d_1} = (4,-3,1)$, $\;\mathbf{d_2} = (-3,0,2)$

$$\mathbf{d_1}\cdot\mathbf{d_2} = 4(-3)+(-3)(0)+1(2) = -12+0+2 = -10 \quad \textbf{M1}$$

$$|\mathbf{d_1}| = \sqrt{16+9+1} = \sqrt{26}, \qquad |\mathbf{d_2}| = \sqrt{9+0+4} = \sqrt{13} \quad \textbf{M1}$$

$$\cos\theta = \frac{|\mathbf{d_1}\cdot\mathbf{d_2}|}{|\mathbf{d_1}||\mathbf{d_2}|} = \frac{10}{\sqrt{26}\cdot\sqrt{13}} = \frac{10}{\sqrt{338}} \quad \textbf{A1}$$

$$\boxed{\theta = \arccos\!\left(\tfrac{10}{\sqrt{338}}\right) \approx 57.0°} \quad \textbf{A1}$$

## Question 5 &nbsp;&nbsp; (12 marks)

$\mathbf{d_1} = (1,4,5)$, $\;\mathbf{d_2} = (3,a,b)$

### Part (a) — 2 marks

$\mathbf{d_1}\cdot\mathbf{d_2} = 0$: &emsp; **M1**

$$3+4a+5b = 0 \implies \boxed{4a+5b = -3} \quad \textbf{A1}$$

### Part (b) — 8 marks

Set lines equal — three equations:

| | Equation |
|--|----------|
| **i** | $4+s = -3+3t \implies s-3t = -7$ &emsp; **(i)** |
| **j** | $1+4s = 1+at \implies 4s = at$ &emsp; **(ii)** |
| **k** | $1+5s = -6+bt \implies 5s-bt = -7$ &emsp; **(iii)** |

**M1 M1** (setting up two equations)

From (ii): $s = \dfrac{at}{4}$. Substitute into (i): &emsp; **M1**

$$\frac{at}{4}-3t = -7 \implies t\!\left(\frac{a-12}{4}\right) = -7 \implies t = \frac{-28}{a-12}$$

Substitute $s = at/4$ into (iii): &emsp; **M1**

$$\frac{5at}{4}-bt = -7 \implies t\!\cdot\!\frac{5a-4b}{4} = -7 \implies t = \frac{-28}{5a-4b}$$

Equate: $a-12 = 5a-4b \implies -4a+4b = 12 \implies b = a+3$ &emsp; **A1**

Solve simultaneously with $4a+5b = -3$: &emsp; **M1**

$$4a+5(a+3) = -3 \implies 9a = -18 \implies \boxed{a = -2,\quad b = 1} \quad \textbf{A1 A1}$$

### Part (c) — 2 marks

$s = at/4 = (-2)t/4 = -t/2$. From (i): $-t/2-3t = -7 \implies t = 2,\;s = -1$ &emsp; **M1**

$$\text{Point} = (4-1,\;1-4,\;1-5) = \boxed{(3,\,-3,\,-4)} \quad \textbf{A1}$$
