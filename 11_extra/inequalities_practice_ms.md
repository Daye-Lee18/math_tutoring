## Question 1 &nbsp;&nbsp; (3 marks)

$|2x-1| = |x+3|$ — square both sides: &emsp; **M1**

$$(2x-1)^2 = (x+3)^2$$

$$4x^2 - 4x + 1 = x^2 + 6x + 9 \implies 3x^2 - 10x - 8 = 0$$

$$(3x+2)(x-4) = 0 \quad \textbf{M1}$$

$$\boxed{x = -\frac{2}{3} \quad \text{or} \quad x = 4} \quad \textbf{A1}$$

## Question 2 &nbsp;&nbsp; (4 marks)

$|3x-1| < 2|x+2|$ — square both sides (both sides are moduli, always $\geq 0$): &emsp; **M1**

$$(3x-1)^2 < 4(x+2)^2$$

$$9x^2 - 6x + 1 < 4x^2 + 16x + 16 \implies 5x^2 - 22x - 15 < 0 \quad \textbf{M1}$$

$$(5x+3)(x-5) < 0 \quad \textbf{M1}$$

Roots: $x = -\dfrac{3}{5}$ and $x = 5$. Upward parabola, negative between roots:

$$\boxed{-\frac{3}{5} < x < 5} \quad \textbf{A1}$$

## Question 3 &nbsp;&nbsp; (4 marks)

$|x-2k| > 3|x+k|$, $k > 0$ — square both sides: &emsp; **M1**

$$(x-2k)^2 > 9(x+k)^2$$

$$x^2 - 4kx + 4k^2 > 9x^2 + 18kx + 9k^2 \implies 8x^2 + 22kx + 5k^2 < 0 \quad \textbf{M1}$$

$$(2x+5k)(4x+k) < 0 \quad \textbf{M1}$$

Roots: $x = -\dfrac{5k}{2}$ and $x = -\dfrac{k}{4}$.

Since $k > 0$: $-\dfrac{5k}{2} < -\dfrac{k}{4}$. Upward parabola, negative between roots:

$$\boxed{-\frac{5k}{2} < x < -\frac{k}{4}} \quad \textbf{A1}$$

## Question 4 &nbsp;&nbsp; (4 marks)

**Sketch:** &emsp; **M1**

- $y = |x+1|$: vertex $(-1, 0)$, y-intercept $= 1$, V-shape
- $y = 3-x$: y-intercept $= 3$, x-intercept $= 3$, slope $= -1$

**Find intersection:** right branch $y = x+1$ with $y = 3-x$: &emsp; **M1**

$$x + 1 = 3 - x \implies 2x = 2 \implies x = 1, \quad y = 2 \quad \textbf{A1}$$

Left branch $y = -(x+1)$ with $y = 3-x$:

$$-(x+1) = 3-x \implies -1 = 3 \quad \text{(no solution — lines are parallel)}$$

**Read from graph:** $|x+1| < 3-x$ where the V-graph lies below the straight line:

$$\boxed{x < 1} \quad \textbf{A1}$$

## Question 5 &nbsp;&nbsp; (5 marks)

**Sketch:** &emsp; **M1**

- $y = |2x-4|$: vertex $(2,0)$, y-intercept $= 4$, V-shape
- $y = x+1$: y-intercept $= 1$, x-intercept $= -1$

**Find intersections:**

Right branch $y = 2x-4$ with $y = x+1$: &emsp; **M1**

$$2x-4 = x+1 \implies x = 5, \quad y = 6 \quad \textbf{A1}$$

Left branch $y = -(2x-4) = 4-2x$ with $y = x+1$: &emsp; **M1**

$$4-2x = x+1 \implies 3 = 3x \implies x = 1, \quad y = 2 \quad \textbf{A1}$$

**Read from graph:** $|2x-4| > x+1$ where V-graph lies above the line:

$$\boxed{x < 1 \quad \text{or} \quad x > 5} \quad \textbf{A1}$$

## Question 6 &nbsp;&nbsp; (6 marks)

### Part (a) — 3 marks

**Sketch:** &emsp; **M1**

- $y = |2x+6|$: vertex $(-3, 0)$, y-intercept $= 6$, V-shape &emsp; **B1**
- $y = x+8$: y-intercept $= 8$, x-intercept $= -8$ &emsp; **B1**

### Part (b) — 3 marks

Right branch $y = 2x+6$ with $y = x+8$: &emsp; **M1**

$$2x+6 = x+8 \implies x = 2, \quad y = 10 \quad \textbf{A1}$$

Left branch $y = -(2x+6) = -2x-6$ with $y = x+8$:

$$-2x-6 = x+8 \implies -14 = 3x \implies x = -\frac{14}{3}, \quad y = \frac{10}{3} \quad \textbf{A1}$$

**Read from graph:** $|2x+6| > x+8$ where V-graph lies above the line:

$$\boxed{x < -\frac{14}{3} \quad \text{or} \quad x > 2} \quad \textbf{A1}$$
