Q1 [13 marks]

f(x) = (x⁴ + x³ − 5x² − 9) / (x² + x − 6)

(a) 대수 나눗셈으로 f(x) = x² + A + B/(x+C) 임을 증명 [5]

먼저 x² + x − 6 = (x+3)(x−2) 인수분해.

다항식 나눗셈:

x⁴ + x³ − 5x² + 0x − 9  ÷  (x² + x − 6)
= x²  → x²(x² + x − 6) = x⁴ + x³ − 6x²
나머지: x² − 9

x² − 9  ÷  (x² + x − 6)
= 1    → 1·(x² + x − 6) = x² + x − 6
나머지: −x − 3

따라서: f(x) = x² + 1 + (−x−3)/((x+3)(x−2)) = x² + 1 + (−1)(x+3)/((x+3)(x−2)) = x² + 1 − 1/(x−2) = x² + 1 + (−1)/(x + (−2))

→ A = 1, B = −1, C = −2

  • M1: 올바른 나눗셈 방법 수행
  • A1: 몫 x² + 1 도출
  • M1: 나머지 분수 처리 (−x−3)/((x+3)(x−2))
  • A1: 약분 후 −1/(x−2)
  • A1: A=1, B=−1, C=−2 명시

(b) f(x) = 0이 정확히 하나의 실근을 가짐을 그래프로 증명 [3] f(x) = 0 → x² + 1 = 1/(x−2) (단, x ≠ 2)

y = x² + 1 (위로 볼록 포물선, 최솟값 (0,1))과 y = 1/(x−2) (직각쌍곡선, x=2 점근선)를 같은 축에 그림.

  • B1: y = x² + 1 올바른 스케치
  • B1: y = 1/(x−2) 올바른 스케치 (x=2 점근선 포함)
  • B1: 두 그래프가 x > 2 영역에서 정확히 1번 교차함 → 실근 1개 결론

(c) 반복식으로 f(x) = 0의 근을 3 유효숫자로 구하기 [5]

시작값 결정 (부호변환): f(2.1) = (2.1)² + 1 − 1/(2.1−2) = 4.41 + 1 − 10 = −4.59 < 0 f(2.5) = 6.25 + 1 − 1/0.5 = 5.25 > 0 → 근 ∈ (2.1, 2.5)

반복식: x_{n+1} = 2 + 1/(x_n² + 1), 시작값 x₀ 선택 (예: x₀ = 2.3)

수렴값: α ≈ 2.17 (3 유효숫자)

정당화: f(2.165) < 0, f(2.175) > 0 → 부호 변환 → α ∈ (2.165, 2.175) → α = 2.17 (3 s.f.)

  • M1: 적절한 시작값 및 부호 변환으로 근 위치 확인
  • M1: 반복식 적용, 4회 이상 반복
  • A1: 수렴값 ≈ 2.17
  • M1: 3 유효숫자 정당화 방법 (f(2.165), f(2.175) 부호 검사)
  • A1: 정당화 완료

Q2 [12 marks]

(a) π − 3 arccos θ = 0 풀기 [2] arccos θ = π/3 → θ = cos(π/3) = 1/2

  • M1: arccos θ = π/3 도출
  • A1: θ = 1/2

(b) y = arccos(x−1) (0≤x≤2)와 y = √(x+2) (x≥−2) 스케치 [5]

y = arccos(x−1): - 정의역: 0 ≤ x ≤ 2 - (0, π): x=0에서 arccos(−1) = π - (1, π/2): x=1에서 arccos(0) = π/2 - (2, 0): x=2에서 arccos(1) = 0 - 감소 곡선

y = √(x+2): - 정의역: x ≥ −2 - (−2, 0): x=−2에서 y=0 - (0, √2): y-절편 - 증가 곡선

  • B1: arccos 곡선 감소 형태
  • B1: (0, π) 및 (2, 0) 표시
  • B1: √(x+2) 증가 형태
  • B1: (−2, 0) 및 (0, √2) 표시
  • B1: 두 곡선 교점 1개 (0 < α < 1 부근)

(c) 0 < α < 1 임을 증명 [3] g(x) = arccos(x−1) − √(x+2) 정의

g(0) = arccos(−1) − √2 = π − √2 ≈ 3.14 − 1.41 > 0 g(1) = arccos(0) − √3 = π/2 − √3 ≈ 1.57 − 1.73 < 0

g 연속, g(0) > 0, g(1) < 0 → 중간값 정리에 의해 α ∈ (0, 1)이 존재

  • M1: g(x) = arccos(x−1) − √(x+2) 설정
  • A1: g(0) = π − √2 > 0 명시
  • A1: g(1) = π/2 − √3 < 0 명시 → 결론

(d) 반복식으로 α를 소수 3자리까지 구하기 [4] x_{n+1} = 1 + cos(√(x_n + 2)), x₀ = 1 (문제 조건)

x₁ = 1 + cos(√3) ≈ 1 + cos(1.732) ≈ 0.8226 x₂ ≈ 0.8786, x₃ ≈ 0.8726, …

수렴값: α ≈ 0.875 (소수 3자리)

정당화: f(0.8745) > 0, f(0.8755) < 0 → α ∈ (0.8745, 0.8755) → α = 0.875 (3 d.p.)

  • M1: x₀ = 1 사용, 반복식 적용
  • M1: 4회 이상 반복
  • A1: α ≈ 0.875
  • A1: 소수 3자리 정당화 (0.8745 < α < 0.8755 부호 검사)

Q1 — 12/13

파트 정답 학생 답 점수 비고
(a) A=1, B=−1, C=−2 A=1, B=−1, C=−2 5/5 ✓ 합성제법(synthetic division)과 긴 나눗셈 병용, 정확
(b) 정확히 1 교점 스케치 및 1개 교점 결론 3/3 ✓ y=x²+1, y=1/(x−2) 모두 올바른 형태
(c) α=2.17 (3 s.f.) 2.1745 (3 s.f.로 기재) 4/5 수렴값 올바름. 단, 3 유효숫자 정당화 오류: 학생은 f(2.1744), f(2.1746) 부호를 확인했으나, 3 유효숫자(2.17)를 정당화하려면 f(2.165) < 0, f(2.175) > 0 확인 필요. 또한 최종 답을 “2.1745”로 적었는데 이는 5 유효숫자임.

⚠️ Q1(c) 정당화 방법 오류: “3 significant figures” 답안은 2.17이며, 정당화는 [2.165, 2.175] 구간에서 부호 변화를 보여야 함. 학생은 [2.1744, 2.1746]에서 부호 확인 → 이는 소수 4자리 정당화. “3 s.f.”와 “3 d.p.”(소수점 3자리) 혼동 주의.

Q2 — 8/12

파트 정답 학생 답 점수 비고
(a) θ = 1/2 θ = 1/2 2/2 ✓ cos(π/3)=1/2 정확
(b) 스케치 arccos 감소, √(x+2) 증가, 교점 표시 대략 올바른 형태 3/5 arccos 곡선 감소 형태 ✓. 그러나 (0, π), (2, 0) 좌표 표기 불명확. √(x+2) 형태 ✓. 교점 위치는 표시됨. B1×3 획득
(c) g(0)=π−√2>0, g(1)=π/2−√3<0 g(0)=π−√2>0 ✓, g(1)=π/2−√3<0 ✓ 3/3 ✓ 완벽한 IVT 논증
(d) α=0.875 (3 d.p.) α=0.875 ✓, 단 x₀=0.5 사용 2/4 최종값 0.875 정확. 단, 문제 조건 x₀=1 을 사용하지 않고 x₀=0.5 사용 → M1 감점. 정당화 구간 [0.8752, 0.8755]으로 3 d.p. 정당화 불완전 (0.8745 < α < 0.8755 필요).

⚠️ Q2(d) x₀ 오류: 문제가 명시적으로 x₀ = 1을 요구했으나 학생은 x₀ = 0.5 사용. 우연히 같은 수렴값에 도달했으나 시작값 오류로 M1 감점. ⚠️ Q2(d) 정당화 간격: “3 decimal places” 정당화는 f(0.8745)와 f(0.8755)의 부호 변화 확인 필요. 학생의 구간은 너무 좁음.

피드백 요약

잘한 점: - Q1(a) 대수 나눗셈: 합성제법 + 긴 나눗셈 병용하여 A=1, B=−1, C=−2 완벽 도출 - Q1(b) 그래프 증명: 두 그래프로 유일 교점 논증 명확 - Q2(a) arccos 방정식: θ=1/2 정확 - Q2(c) IVT 논증: g(0), g(1) 부호 확인 + 결론 완벽

개선 필요: 1. 유효숫자 vs 소수자리 구분: “3 significant figures”는 2.17이고 정당화 구간은 [2.165, 2.175]. “3 decimal places”는 0.875이고 정당화 구간은 [0.8745, 0.8755]. 학생이 두 개념을 혼용함. 2. 반복식 시작값: 문제에서 x₀ 값이 주어지면 반드시 그것을 사용 (Q2(d): x₀=1 요구). 3. 스케치 좌표 표기: 특징점 (0, π), (2, 0) 등을 스케치에 명확히 레이블링.
