채점지 — 01 Functions A

학생: 이상민 | 총점: 22/37

MARK SCHEME (정답 및 배점)

Q1 [6 marks]

f : x → 2 − x², g : x → 3x/(2x−1)

(a) fg(2) [2] - M1: g(2) = 3(2)/(2·2−1) = 6/3 = 2 - A1: fg(2) = f(2) = 2 − 2² = −2

(b) gf(x) = 1/2 [4] - M1: gf(x) = g(2−x²) = 3(2−x²)/(2(2−x²)−1) = (6−3x²)/(3−2x²) - M1: (6−3x²)/(3−2x²) = 1/2 → 12−6x² = 3−2x² - M1: 4x² = 9 → x² = 9/4 - A1: x = ±3/2

Q2 [5 marks]

f(x) = 2 + ln(3x − 2), x > 2/3

(a) ff(1) [2] - M1: f(1) = 2 + ln(3·1 − 2) = 2 + ln1 = 2 - A1: ff(1) = f(2) = 2 + ln(3·2 − 2) = 2 + ln4 = 2 + 2ln2

(b) f⁻¹(x) [3] - M1: y = 2 + ln(3x−2) → y−2 = ln(3x−2) → e^(y−2) = 3x−2 - M1: x = (e^(y−2) + 2) / 3 - A1: f⁻¹(x) = (e^(x−2) + 2) / 3

Q3 [12 marks]

f(x) = x² − 2x + 5, x ≥ 1

(a) Completed square [2] - B2: (x − 1)² + 4 (a = −1, b = 4)

(b) Range [1] - B1: f(x) ≥ 4

(c) f⁻¹(x) [3] - M1: y = (x−1)² + 4 → y−4 = (x−1)² - M1: x−1 = √(y−4) (positive root since x ≥ 1) - A1: f⁻¹(x) = √(x−4) + 1, domain x ≥ 4

(d) Two transformations: y = f⁻¹(x) → y = √x [2] f⁻¹(x) = √(x−4) + 1 - B1: Translation of 4 units in the negative x-direction (left) - B1: Translation of 1 unit in the negative y-direction (down) (These may be combined as translation by vector (−4, −1))

(e) Normal to y = f⁻¹(x) at x = 8 [4] - M1: f⁻¹(8) = √(8−4) + 1 = 3 → point (8, 3) - M1: (f⁻¹)‘(x) = 1/(2√(x−4)) → (f⁻¹)’(8) = 1/(2·2) = 1/4 - M1: Normal gradient = −4 - A1: y − 3 = −4(x − 8) → y = −4x + 35

Q4 [13 marks]

f(x) = 3 − x², x ≥ 0; g(x) = 8/(3−x), x ≠ 3

(a) Range of f [1] - B1: f(x) ≤ 3

(b) Sketch y = f(x) and y = f⁻¹(x) [3] - B1: f(x) correct shape (decreasing from (0,3)) - B1: f⁻¹(x) reflection in y = x - B1: Correct intercepts and reflection line y = x shown

(c) f⁻¹(x) and domain [4] - M1: y = 3 − x² → x² = 3−y → x = √(3−y) (x ≥ 0) - A1: f⁻¹(x) = √(3−x) - B1: Domain: x ≤ 3 - B1: (Or state domain is range of f: −∞ < x ≤ 3)

(d) fg(−3) [2] - M1: g(−3) = 8/(3−(−3)) = 8/6 = 4/3 - A1: f(4/3) = 3 − (4/3)² = 3 − 16/9 = 11/9

(e) f⁻¹(x) = g(x) [3] - M1: √(3−x) = 8/(3−x); let u = 3−x → √u = 8/u → u^(3/2) = 8 - M1: u = 8^(2/3) = 4 → 3−x = 4 - A1: x = −1

Q5 [5 marks (문제지에 Q4 이후 배치)]

Note: Q5 (f(x) = 4 − ln3x, g(x) = e^(2−x))는 Transformation 과제(02)에도 일부 동일 문제가 존재함. 여기서는 Functions A 지 기준으로 채점.

(a) f(x) = 0 [2] - M1: 4 − ln3x = 0 → ln3x = 4 → 3x = e⁴ - A1: x = e⁴/3

(b) Sketch y = f(x) [2] - B1: Correct shape (decreasing, defined for x > 0) - B1: x-intercept at (e⁴/3, 0), asymptote x = 0 shown

(c) f⁻¹(x) [3] - M1: y = 4 − ln3x → ln3x = 4−y → 3x = e^(4−y) - A1: x = e^(4−y)/3 - A1: f⁻¹(x) = (1/3)e^(4−x)

(d) Show fg(x) = x + a − lnb [3] - M1: g(x) = e^(2−x); fg(x) = f(e^(2−x)) = 4 − ln(3e^(2−x)) - M1: = 4 − (ln3 + ln(e^(2−x))) = 4 − ln3 − (2−x) = x + 2 − ln3 - A1: a = 2, b = 3
