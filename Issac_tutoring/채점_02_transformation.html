Q1 [9 marks]

원래 곡선: 최댓값 (−3, 2), 최솟값 (2, −4)

(a)(i) y = f(|x|) [3.5] |x| 변환: x ≥ 0 부분을 y축에 대해 반사 - B1: x ≥ 0 부분은 원래와 동일 → 최솟값 (2, −4) 유지 - B1: x < 0 부분은 x > 0 부분의 반사 → 최솟값 (−2, −4) 추가 - B1: y축 대칭인 형태, 정류점 좌표 (±2, −4) 명시 - (x = 0에서의 점은 f(0)의 값으로 표시)

(a)(ii) y = 3f(2x) [3.5] - x방향 1/2 배율 (x → x/2), y방향 3배 확대 - 원래 최댓값 (−3, 2) → (−3/2, 6) - 원래 최솟값 (2, −4) → (1, −12) - B1: 형태 올바름 - B1: 최댓값 (−3/2, 6) 명시 - B1: 최솟값 (1, −12) 명시

(b) y = a + f(x + b), 최솟값이 원점 [2] - M1: 원래 최솟값 (2, −4)를 (0, 0)으로 이동 - x: x + b = 2 → b = 2 (2칸 왼쪽 이동) - y: a + (−4) = 0 → a = 4 - A1: a = 4, b = 2

Q2 [10 marks]

y = 2 + sec(x − π/6), 0 ≤ x ≤ 2π

(a) 스케치 [5] 점근선: cos(x − π/6) = 0 → x − π/6 = π/2, 3π/2 → x = 2π/3, x = 5π/3

정류점: - x − π/6 = 0 → x = π/6: sec = 1, y = 3 → (π/6, 3) 최솟값 - x − π/6 = π → x = 7π/6: sec = −1, y = 1 → (7π/6, 1) 최댓값 - x − π/6 = 2π → x = 13π/6 > 2π (범위 밖)

  • B1: 전체적인 sec 형태 (두 가지)
  • B1: 정류점 (π/6, 3) 정확
  • B1: 정류점 (7π/6, 1) 정확
  • B1: 점근선 x = 2π/3 정확
  • B1: 점근선 x = 5π/3 정확

(b) y = 0인 x 좌표 (0 ≤ x ≤ 2π) [5] 2 + sec(x − π/6) = 0 → sec(x − π/6) = −2 → cos(x − π/6) = −1/2 → x − π/6 = 2π/3 or 4π/3 → x = 5π/6 or x = 3π/2

  • M1: sec(x−π/6) = −2
  • M1: cos(x−π/6) = −1/2
  • M1: 두 각도 2π/3, 4π/3 도출
  • A1: x = 5π/6
  • A1: x = 3π/2

Q3 [12 marks]

y = 4a² − x² 와 y = |2x − a| (a > 0)

(a) 두 그래프 스케치 및 좌표축 교점 in terms of a [6] y = 4a² − x²: - x-절편: ±2a, y-절편: 4a² - 꼭짓점: (0, 4a²) 최댓값

y = |2x − a|: - V-형, 꼭짓점: (a/2, 0) - x-절편: a/2 (단, y = 0에서) - y-절편: |−a| = a

  • B1: 포물선 형태 올바름
  • B1: 포물선 x-절편 ±2a, y-절편 4a² 표시
  • B1: V자 형태 올바름
  • B1: V자 꼭짓점 (a/2, 0) 표시
  • B1: y-절편 a 표시
  • B1: 교점 좌표 in terms of a (2개 교점)

(b) 4 − x² = |2x − 1| 의 정확한 해 [6]

경우 1: 2x − 1 ≥ 0 (x ≥ 1/2) 4 − x² = 2x − 1 → x² + 2x − 5 = 0 → x = (−2 ± √20)/2 = −1 ± √5 x = −1 + √5 ≈ 1.236 ✓ (x ≥ 1/2 만족) x = −1 − √5 ✗ (x ≥ 1/2 불만족)

경우 2: 2x − 1 < 0 (x < 1/2) 4 − x² = −(2x − 1) = 1 − 2x → x² − 2x − 3 = 0 → (x − 3)(x + 1) = 0 → x = 3 or x = −1 x = 3 ✗ (x < 1/2 불만족) x = −1 ✓ (x < 1/2 만족)

정답: x = −1 + √5 and x = −1

  • M1: 경우 분리
  • A1: 경우 1 이차방정식 수립
  • A1: x = −1 + √5
  • A1: 경우 2 이차방정식 수립
  • A1: x = −1 (검증 포함)
  • B1: 유효하지 않은 해 기각

Q4 [14 marks]

f(x) = 2 − √(x+9), x ≥ −9; 좌표축 교점 (a, 0) and (0, b)

(a)(i) y = f⁻¹(x) [3] f의 교점 (a, 0), (0, b) → f⁻¹의 교점 (b, 0), (0, a) - B1: (0, a) y-절편 표시 - B1: (b, 0) x-절편 표시 - B1: 그래프 형태 올바름 (증가 함수, 감소 f의 역)

(a)(ii) y = 2f(3x) [3] x방향 1/3 배율, y방향 2배 - 원래 (a, 0) → (a/3, 0) - 원래 (0, b) → (0, 2b) - B1: (a/3, 0) x-절편 표시 - B1: (0, 2b) y-절편 표시 - B1: 그래프 형태 올바름

(b) a와 b의 값 [3] f(a) = 0: 2 − √(a+9) = 0 → √(a+9) = 2 → a+9 = 4 → a = −5 f(0) = 2 − √9 = 2 − 3 = b = −1

  • M1: f(a) = 0 설정
  • A1: a = −5
  • A1: b = −1

(c) f⁻¹(x) 표현식과 정의역 [5] y = 2 − √(x+9) → 2−y = √(x+9) → (2−y)² = x+9 → x = (2−y)²−9 f⁻¹(x) = (2−x)² − 9 or (x−2)² − 9

정의역: f의 치역 = f⁻¹의 정의역 f(x) = 2 − √(x+9) → 최댓값 f(−9) = 2, x가 클수록 −∞ → 치역: x ≤ 2 → 정의역: x ≤ 2

  • M1: y = f(x) → x에 대해 풀기
  • M1: (2−y)² = x+9
  • A1: f⁻¹(x) = (2−x)²−9 (또는 동치 형태)
  • B1: 정의역 x ≤ 2 명시
  • B1: 이유 설명 (f의 치역)
