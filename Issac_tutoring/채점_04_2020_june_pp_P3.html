상세 채점

Q1 · 삼각방정식 [5/5] ✓

2cos2x = 7cosx, 0 ≤ x < 360°

학생 풀이: - cos2x = 2cos²x − 1 대입 → 4cos²x − 7cosx − 2 = 0 (M1 ✓) - (4c+1)(c−2) = 0 인수분해 → cosx = −1/4, 2 (M1 A1 ✓) - cosx = 2는 범위 초과 제외, cosx = −1/4 선택 (dM1 ✓) - x = 104.5°, 255.5° (A1 ✓)

Q2 · 지수·로그 모델 [6/6] ✓

log₁₀N = 0.0646t + 1.478

(a) N = ab^t 형태 [4/4] - N = 10^(0.0646t + 1.478) = 10^1.478 × (10^0.0646)^t (M1 A1 ✓) - a = 30, b = 1.16 도출, N = 30 × 1.16^t (dM1 A1 ✓)

(b) t = 30일 때 N [2/2] - N = 30 × 1.16^30 ≈ 2600 mm² (M1 A1 ✓)

Q3 · 미분·치역 [7/7] ✓

f(x) = (2x+3)/√(4x−1), x > 1/4

(a) f’(x) [4/4] - 몫의 법칙 올바르게 적용: 분자 (4x−1)^(1/2)·2 − (2x+3)·2(4x−1)^(-1/2) (M1 A1 ✓) - (4x−1)^(1/2) 곱해 정리 → f'(x) = 4(x−2)/(4x−1)^(3/2) = (4x−8)/(4x−1)^(3/2) (dM1 A1 ✓)

(b) range of f [3/3] - f'(x) = 0 → x = 2 (M1 ✓) - f(2) = 7/√7 = √7 → dM1 A1 ✓ - range: f ≥ √7 ✓

Q4 · 합성함수·변환 [8/8] ✓

f(x) = 21 − 2|2−x|, x ≥ 0

(a) ff(6) [2/2] - f(6) = 21 − 2|2−6| = 21 − 8 = 13 (M1 ✓) - ff(6) = f(13) = 21 − 2|2−13| = 21 − 22 = −1 (A1 ✓)

(b) f(x) = 5x → x = 25/7 [2/2] - x > 2인 경우: 21 − 2(x−2) = 5x → 7x = 25 → x = 25/7 (M1 A1 ✓)

(c) f(x) = k의 근이 2개인 k 범위 [2/2] - 17 ≤ k < 21 (M1 A1 ✓)

(d) a, b 값 [2/2] - 꼭짓점 (2, 21) → (6, 3) 변환 이용 - 21 × a = 3 → a = 1/7, 2 + b = 6 → b = 4 (B1 B1 ✓)

Q5 · 삼각 항등식·적분 [6/8] ⚠️

(a) sin3x ≡ 3sinx − 4sin³x 증명 [4/4] ✓ - sin3x = sin(2x+x) = sin2x·cosx + cos2x·sinx (M1 ✓) - = 2sinx·cos²x + (1−2sin²x)sinx (M1 ✓) - = sinx(2−2sin²x + 1−2sin²x) → = sinx(3−4sin²x) (ddM1 ✓) - = 3sinx − 4sin³x ∴ 증명 완료 (A1* ✓)

(b) ∫₀^(π/3) sin³x dx [2/4] ⚠️ −2점

마크 기준 결과
M1 (a) 이용해 sin³x = 3/4 sinx − 1/4 sin3x ✓
dM1 A sinx + B sin3x → a cosx + b cos3x 형태 ✓
A1 −3/4 cosx + 1/12 cos3x 正확한 계수 ✗ (1/4 cos3x 로 기재, 1/12가 맞음)
A1 최종 답 5/24 ✗ (5/8로 오답)

오류 설명: ∫(−1/4 sin3x)dx = (1/12)cos3x 인데, 학생이 (1/4)cos3x로 적분 상수를 빠뜨림(3으로 나누기 미적용). 따라서 최종 답 5/8 ≠ 5/24.

정확한 계산: [−3/4 cosx + 1/12 cos3x]₀^(π/3) = (−3/8 − 1/12) − (−3/4 + 1/12) = −11/24 + 8/12 = 5/24

Q6 · 수치해석 [6/9] ⚠️

(a) P점의 x좌표 [3/3] ✓ - 5e^(x−1) + 3 = 18 → e^(x−1) = 3 → x = ln3 + 1 = ln3e (M1 A1 A1 ✓)

(b) α = 1.134 (3dp) 증명 [1/3] ⚠️ −2점

마크 기준 결과
B1 적절한 함수 정의: f(x) = x² + 5e^(x−1) − 7 ✓
M1 x = 1.1335, x = 1.1345 대입 (또는 더 좁은 구간) ✗
A1 올바른 값, 부호 변화 이유, 결론 ✗ (M1 미획득으로 종속)

오류 설명: 학생이 [1.133, 1.135] 구간을 사용했으나, 3dp 정밀도 증명에는 [1.1335, 1.1345] (또는 더 좁은 구간)이 필요. [1.133, 1.135]는 너무 넓어 α = 1.134를 3dp로 확정할 수 없음.

올바른 접근: f(1.1335) ≈ −0.003 < 0, f(1.1345) ≈ +0.004 > 0, 부호 변화 + 연속 → α = 1.134 (3dp)

(c) x₂, β 값 [2/3] ⚠️ −1점

마크 기준 결과
M1 x₁ = −3 대입 시도 ✓
A1 x₂ = awrt −2.628388 ✓ (−2.628388 ✓)
A1 β = awrt −2.620330 ✗ (학생: −2.620535)

오류 설명: 학생이 x₃를 β로 정의했으나 x₃ ≈ −2.62054 ≠ β = −2.620330. β는 반복수열의 수렴값(극한)으로, 더 많은 반복이 필요: - x₃ ≈ −2.62054 - x₄ ≈ −2.62031 - x₅ ≈ β ≈ −2.620330 (6dp)

Q7 · R형 변환·응용 [9/9] ✓

(a) cosx + 4sinx = R cos(x−α) [3/3] - R = √(1+16) = √17 (B1 ✓) - tanα = 4 → α = 1.326 rad (M1 A1 ✓)

(b) 최소 높이 [2/2] - H_min = 24/(3+√17) ≈ 3.37 m (M1 A1 ✓)

(c) H = 10일 때 t [4/4] - cos(t/2 − 1.326) = −0.6/√17 (M1 A1 ✓) - t/2 − 1.326 = arccos(−0.6/√17) ≈ 1.7168 (M1 ✓) - t = 2(1.7168 + 1.326) ≈ 6.09 (A1 ✓)

Q8 · 미분(적·몫의 법칙, 음함수) [9/9] ✓

(i)(a) g’(x) [2/2] - g'(x) = 3e^(3x)sec2x + 2e^(3x)sec2x·tan2x = e^(3x)sec2x(3+2tan2x) (M1 A1 ✓)

(i)(b) 정류점 x좌표 [3/3] - e^(3x) ≠ 0, sec2x ≠ 0 → 3 + 2tan2x = 0 → tan2x = −1.5 (M1 dM1 ✓) - x = −0.491 (A1 ✓)

(ii) dy/dx = eˣ/f(x) 증명 [4/4] - x = ln(siny) → dx/dy = cosy/siny → dy/dx = siny/cosy = tany (B1 ✓) - cosy = √(1−sin²y) = √(1−e^(2x)) 이용 (M1 ✓) - dy/dx = siny/cosy = e^x/√(1−e^(2x)) (dM1 A1 ✓)

Q9 · 대수 나눗셈·미적분 [14/14] ✓

(a) P=2, Q=5 [4/4] - 조립제법으로 (x+3) 인수 분리 후 x−4로 나눗셈: 몫 x²+2, 나머지 5 (M1 M1 ✓) - P = 2, Q = 5 (A1 A1 ✓)

(b) x=2에서 접선 방정식 [5/5] - g'(x) = 2x − 5/(x−4)² (M1 A1 ✓) - g'(2) = 4 − 5/4 = 11/4 (M1 ✓) - g(2) = 4+2−5/2 = 7/2, 접선: y = 11/4·x − 2 (dM1 A1 ✓)

(c) 넓이 R (정확값) [5/5] - ∫₀²(x²+2+5/(x−4))dx = [1/3·x³+2x+5ln|x−4|]₀² (M1 A1 ✓) - = (8/3+4+5ln2) − (5ln4) → = 20/3 − 5ln2 (dM1 ddM1 A1 ✓)
